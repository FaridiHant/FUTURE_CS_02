🛡️ FUTURE_CS_02 — Phishing Email Detection & Awareness Report
Cyber Security Internship | Future Interns 2026
📌 Project Overview

This project presents a detailed analysis of a phishing email sample conducted during the Future Interns Cyber Security Internship (2026).

The goal was to simulate a real-world SOC (Security Operations Center) task by identifying phishing indicators, assessing risk, and creating awareness for users and organizations.

🎯 Objectives
Identify phishing characteristics in real email samples
Analyze email headers (SPF, DKIM, DMARC)
Investigate suspicious links and sender behavior
Classify threat severity
Document findings in a professional report
Promote cybersecurity awareness

📧 Email Sample Details
Impersonated Organization: Chase Bank (JPMorgan Chase)
Actual Sender: merciayanmact@hotmail.com
Classification: 🔴 Phishing
Risk Level: 🔴 High
Date Analyzed: April 13, 2026

🧰 Tools & Techniques
Manual Header Analysis
Reviewed SPF, DKIM, and DMARC authentication results
MXToolbox Email Header Analyzer
Verified email routing and server authenticity
Browser Developer Tools
Inspected hidden and redirected links
EML Viewer
Simulated how the email appears to a victim

🚨 Phishing Indicators Identified
Sender email does not match the claimed organization (high risk)
Generic greeting with no personalization (high risk)
Urgent language threatening account suspension (high risk)
Suspicious shortened or hidden link (high risk)
Email routed through non-corporate servers (medium risk)
